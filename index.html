¿Qué es un límite?

Intuitivamente, el límite de una función describe el valor al que se aproxima f(x) cuando x se acerca a un punto a, sin necesariamente alcanzarlo.

Imagina caminar hacia una puerta pero nunca cruzarla — el límite es esa puerta: el destino al que te acercas, no donde estás.

Definición formal
lim f(x) = L  si y sólo si

para todo ε > 0 existe δ > 0
tal que si 0 < |x − a| < δ,
entonces |f(x) − L| < ε.
